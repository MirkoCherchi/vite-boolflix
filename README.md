## Come Iniziare

1. Clona il repository sul tuo computer:

```bash
git clone https://github.com/tuonome/mirkflix.git
```

2. Entra nella directory del progetto:

```bash
cd mirkflix
```

3. Installa le dipendenze:

```bash
npm install
```

4. Avvia l'applicazione:

```bash
npm run serve
```
